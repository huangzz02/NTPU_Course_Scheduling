# NTPU 通識試排系統

**版本：v2.0.0**  
**作者：黃振彰**  
**網址：** https://huangzz02.github.io/NTPU_Course_Scheduling/

## 專案簡介

這是一個純前端的通識課程試排工具，專為國立臺北大學學生設計。  
幫助你在正式選課前模擬志願順序、檢查時間衝突、避開不想上的時段，並即時確認課程組合是否可行。

本次 **v2.0.0** 改版全面導入「液態玻璃 (Liquid Glass)」介面設計，並重構底層邏輯，大幅提升操作質感與視覺體驗。

目前以電腦顯示較為穩定，雖然有些許做了手機的 RWD 響應式設計，但還不夠全面。  
內建操作導覽、深色模式與一鍵分享功能。

## 主要功能更新

- **全新視覺介面**：採用 CSS 變數系統實作玻璃擬態風格，並全面替換為高品質 SVG 圖示，解決跨裝置顯示不一致問題。
- **進階篩選機制**：除了原有的向度篩選，**新增「星期」篩選**（週一至週五），支援交集搜尋，選課更靈活。
- **互動式新手導覽**：具有步驟教學，引導使用者快速熟悉時段屏蔽與排序功能。
- **檢視模式切換**：提供「方格卡片」與「列表」兩種瀏覽模式，並新增流暢的切換動畫。
- **優化拖曳排序**：志願排序區域支援直覺拖曳，並大幅修正手機版觸控體驗。
- **時段屏蔽網格**：視覺化 5x9 網格，點擊格子即可屏蔽該時段，搜尋結果將自動過濾衝堂課程。
- **智慧評價搜尋**：點擊「評價」按鈕，自動帶入「台北大學 + 課程名稱 + 評價」至 Google 搜尋。
- **即時資訊揭露**：即時統計志願課程數量，並完整顯示剩餘名額、可加簽狀態、授課語言與備註。
- **一鍵分享**：志願順序自動寫入網址參數，「分享」按鈕可複製連結，直接還原當前的排序結果。

## 如何使用

1. **新手導覽**：首次進入可跟隨畫面上的互動教學，快速熟悉介面操作。
2. **時段屏蔽**：若有不能上的時段，直接點擊上方時間網格的格子進行屏蔽。
3. **搜尋課程**：在搜尋欄輸入關鍵字，或利用**向度**與**星期**篩選按鈕找課。
4. **加入志願**：找到課程後，可先點擊「評價」按鈕查詢評價，確認後點擊「＋加入」。
5. **調整排序**：在「志願排序」區域拖曳卡片調整優先順序。
6. **保存分享**：確認沒問題後，點右上角「分享」複製連結備用。
